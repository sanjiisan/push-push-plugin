PushPushGo

== Description ==

PushPushGo enabled web push notifications on your WordPress site. 

Major features in PushPushGo plugins include:

* Automatically add integration code to your WordPress site
* Easy to enable and disable plugin from WordPress site

It's only integration plugin! All features from PushPushGo is available from PushPushGo dashboard.

PS: You'll need an [PushPushGo API Key](https://app.pushpushgo.com/) to use it.

== Installation ==

 1. Install plugin on Your WordPress site.
 2. Create account on PushPushGo.com and generate public API in Profile section.
 3. Create project in PushPushGo.com.
 4. Go to WordPress Admin then enter your [PushPushGo API Key](https://app.pushpushgo.com/).
 5. Select project to Integrate and click "Save changes".
 6. You're done!

 If you have any problem with integration - feel free to contact with us support@pushpushgo.com.
